How to publish on GitHub Pages:
1. Open your repository: bernie8026/story.github.io
2. Upload all files and folders from this package to the root of the repo.
3. Make sure index.html stays at the root.
4. In GitHub -> Settings -> Pages:
   - Source: Deploy from a branch
   - Branch: main
   - Folder: / (root)
5. Save and wait for GitHub Pages to build.
